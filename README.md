  - [Stylelin](https://marketplace.visualstudio.com/items?itemName=stylelint.vscode-stylelint) - css格式化

## 安装使用

- 获取代码

```sh
git clone xxx.git
```

- 安装依赖

```sh
yarn install
```

- 运行

```sh
yarn serve
```

- 打包

```sh
yarn build
```

- 本地预览

```sh
yarn preview
```

- 打包/预览

```sh
yarn build:preview
```
